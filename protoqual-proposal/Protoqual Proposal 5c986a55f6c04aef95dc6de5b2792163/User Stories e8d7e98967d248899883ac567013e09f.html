Later Phase (Recommended)

User Management

  • As a user, I want to be able to register my user account because I need access to the product
  • As a user, I want to be able to login using my email and password because I need to access the product
  • As a user, I want to be able to reset my password because I might forget it
  • As a user, I want to be able to change my password because it may have been compromised
  • As an administrator, I want to be able to reset a user’s password because it is important for managing security
  • As an administrator, I want to be able to register users on their behalf because it may help us manage our account

Onboarding

  • As a user, I want to be able to fill out an onboarding survey that further defines my role because it will be helpful to complete my account and improve my product experience
  • As a user, I want to be able to fill out an onboarding survey that allows my to provide my content preferences because it will be helpful to complete my account and improve my product experience

Dashboard

  • As a user, I want to see my dashboard when I login because it will make it easier for me to navigate the product
  • As a user, I want to see information about my current assignments because it is relevant to the work I am expected to do
  • As a user, I want to be able to see how much progress I’ve made towards my certifications and learning goals because it will be rewarding for me to monitor my progress

Certificates

  • As a user, I want a certificate of completion to display my name because it will feel more personalized to me
  • As a user, I want to be able to print out my certificate because I may want to hang it up or share it with others

Courses

  • As a user, I want to receive a certificate of completion whenever I complete a course because that will encourage me to continue learning
  • As a user, I want to receive recommendations for courses I might enjoy taking because they will be personalized to my educational needs
  • As a user, I want to be presented with occasional opportunities to review what I’ve learned because that will help me retain the knowledge I’ve gained
  • As a user, I want to provide an evaluation for a particular course because it will be helpful for others to know which courses are the most popular
  • As an admin, I want to be able to set learning goals for my team because it will enable them to accomplish more

Testing

  • As a user, I want every course to offer a test because I want to know how well I learned that course material
  • As a user, I want to score a certain level to be able to complete a course because if I haven’t learned the material I’ll need to retake the course
  • As a user, I want to be able to review how I performed on a test because I want to know what I got wrong, what the correct answer is and why that is the correct answer Notifications
  • As a user, I want to be able to receive notifications by email about my course assignments due date because it will help remind me that I have a course to complete
  • As a user, I want to be able to receive notifications by email about any past due assignments because it will help remind me that I have a course to complete
  • As a user, I want to be able to receive email notifications about discussion activity because it may be relevant to my work
  • As a user, I want to receive email notifications about courses I may be interested in taking because they will help me learn about content that may be relevant to me
  • As a user, I want to receive an email notification whenever a new assignment has been given to me because I need to complete the assignment
  • As a user, I want to receive an email notification about an assignment that is due within 48 hours because I will need to complete the assignment soon.
  • As a user, I want to receive email notifications from administrators that have been sent to my group because it will help me stay in communication with my group.
  • As a user, I want to receive email notifications about my progress because it will be helpful to reinforce the progress I’ve made.

Discussion Board

  • As user, I want to be able to post topics because it will be helpful for me to get questions answered and interact with others
  • As a user, I want to be able to search discussions because I may have questions that someone else has previously had
  • As a user, I want to be able to filter discussions by tags because it will help me find discussions that I’m interested in
  • As a user, I want to be able to add tags to my discussion because it will make it easier for others to find my post
  • As a user, I want to be able to change the visibility on my discussion post between my office, my group of offices and public because I may only want certain people to see my discussion

Reporting

  • As an admin, I want to be able to report on learning utilization for my team because it will be important to monitor their progress
  • As an admin, I want to be able to report on patient survey utilization because it will help us monitor our performance
  • As an admin, I want to be able to compare my office performance against other offices and a national benchmark because it will help identify how well our team is performing Surveys
  • As a user, I want to be able to customize surveys because I will need to make changes
  • As a user, I want to be able to send surveys through email or SMS because the patient may have a preference
  • As a user, I want to be able to create and share a QR code for a survey because it will make sharing the survey easier
  • As a patient, I want to be able to know how much is left in a survey because I want to know when I’ll be done
  • As a user, I want patients to have the ability to provide open ended feedback because the patient may have valuable information to share about their experience

Questions

  • Any reason to keep username login? Why not just email?
  • What does ‘lookup group information’ mean? Why is this needed? Technical Architecture
  • Cookie persistence
  • OAuth (sharing ID with another authentication system)
  • Mobile responsiveness (surveys)
